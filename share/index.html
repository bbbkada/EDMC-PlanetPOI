How to use this link

  • If you already have the PlanetPOI plugin installed:
    Simply paste the URL into the PlanetPOI Add dialog.
  • If you do NOT have the plugin:
    Follow the installation instructions below.

Installation requirements

  1. Install Elite Dangerous Market Connector (EDMC)
    https://github.com/EDCD/EDMarketConnector
  2. Install the PlanetPOI plugin:
    https://github.com/bbbkada/EDMC-PlanetPOI
  3. To see POIs as an in-game overlay, you also need EDMCOverlay:
    https://github.com/inorton/EDMCOverlay
